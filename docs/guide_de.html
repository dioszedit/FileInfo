FileInfo

Benutzerhandbuch — wähle eine Datei aus und sieh sofort alles, was über sie bekannt ist: Streams und Codecs bei Videos, Tags und Coverbild bei Musik, EXIF bei Fotos und Spotlight-Metadaten bei allem anderen.

Erste Schritte

Installation

cd FileInfo
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
brew install ffmpeg exiftool

Starten

ModusWie
Aus dem FinderDoppelklick auf FileInfo.command
Aus dem Terminal./run.sh
Kommandozeile (ohne GUI)./run.sh /Pfad/zur/Datei.mkv — gibt die Metadaten im Terminal aus

Die Oberfläche

Das Fenster ist in zwei Bereiche geteilt; der Trenner ist verschiebbar und seine Position wird gespeichert:

  • Dateibaum (links) — startet in deinem Benutzerordner. Mit „Ordner wählen…" kommst du überallhin; externe Laufwerke findest du unter /Volumes.
  • Metadaten-Bereich (rechts) — lädt automatisch bei Auswahl. Oben das Vorschaubild (Foto, Videobild oder Albumcover), darunter Dateiname, Filterfeld und die gruppierten Daten.
  • Filterfeld — grenzt die Liste beim Tippen ein: gib z. B. Sprache ein, und nur Sprachzeilen bleiben übrig.
  • Rechtsklick auf eine Zeile — „Wert kopieren" / „Zeile kopieren".
  • Sprache-Menü — die Sprache der Oberfläche ist umschaltbar (Ungarisch, Englisch, Deutsch, Spanisch, Französisch); wirksam nach einem Neustart.
  • SHA-256-Schaltfläche (unten im Panel) — berechnet die Prüfsumme der ausgewählten Datei im Hintergrund, für jede Dateigröße; das Ergebnis ist auswählbar und in „Metadaten kopieren" enthalten.
  • Dunkel- und Hellmodus — die App folgt automatisch dem Erscheinungsbild des Systems (Systemeinstellungen → Erscheinungsbild) und wechselt live; einen eigenen Schalter gibt es nicht.
  • Ansicht → Versteckte Dateien anzeigen — wie der Finder blendet der Baum Punktdateien standardmäßig aus; hier einschaltbar.

FAQ / Fehlerbehebung

Warum sehe ich bei einem Bild keine EXIF-Daten?

Screenshots und heruntergeladene oder per Messenger erhaltene Bilder enthalten meist kein EXIF — die Dienste entfernen es. Das ist kein Fehler.

Wie erreiche ich einen USB-Stick / ein externes Laufwerk?

📁 Ordner wählen… → das Laufwerk unter „Orte" auswählen oder /Volumes eingeben.

Dateien auf einem Netzlaufwerk laden langsam

Die Geschwindigkeit hängt davon ab, wie schnell die Datei gelesen werden kann. Die Oberfläche friert nie ein, und angezeigt werden immer die Daten der zuletzt ausgewählten Datei.

Der Spotlight-Abschnitt ist bei einer Datei leer

Spotlight indexiert nicht jeden Ort (temporäre Ordner, manche externe Laufwerke). Die übrigen Abschnitte sind davon nicht betroffen.

Das Fenster öffnet sich an seltsamer Stelle / Layout wirkt verschoben

Fenstergröße und Trennerposition werden gespeichert; einmal zurechtziehen, und der nächste Start stellt sie wieder her.
