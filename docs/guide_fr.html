FileInfo

Guide de l'utilisateur — sélectionnez un fichier et voyez instantanément tout ce qu'on en sait : pistes et codecs pour les vidéos, étiquettes et pochette pour la musique, EXIF pour les photos et métadonnées Spotlight pour tout le reste.

Premiers pas

Installation

cd FileInfo
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
brew install ffmpeg exiftool

L'interface

La fenêtre est divisée en deux panneaux ; le séparateur est déplaçable et sa position est mémorisée :

  • Arborescence (à gauche) — démarre dans votre dossier personnel. « Choisir un dossier… » permet d'aller n'importe où ; les disques externes sont dans /Volumes.
  • Panneau de métadonnées (à droite) — se charge automatiquement à la sélection. En haut l'aperçu (photo, image de la vidéo ou pochette), puis le nom du fichier, le filtre et les données groupées.
  • Champ de filtre — réduit la liste au fil de la saisie : tapez p. ex. langue et seules les lignes de langue restent.
  • Clic droit sur une ligne — « Copier la valeur » / « Copier la ligne ».
  • Menu Langue — change la langue de l'interface (hongrois, anglais, allemand, espagnol, français) ; prend effet après un redémarrage.
  • Affichage → Afficher les fichiers cachés — comme le Finder, l'arborescence masque les fichiers cachés par défaut ; activez-les ici.

FAQ / Dépannage

Pourquoi ne vois-je pas d'EXIF sur une image ?

Les captures d'écran et les images téléchargées ou reçues par messagerie ne contiennent généralement pas d'EXIF — les services le suppriment. Ce n'est pas un bug.

Comment accéder à une clé USB / un disque externe ?

📁 Choisir un dossier… → choisissez le disque sous « Emplacements » ou saisissez /Volumes.

Les fichiers sur un lecteur réseau se chargent lentement

La vitesse dépend de la rapidité de lecture du fichier. L'interface ne se fige jamais et les données affichées correspondent toujours au dernier fichier sélectionné.

La section Spotlight est vide pour certains fichiers

Spotlight n'indexe pas tous les emplacements (dossiers temporaires, certains disques externes). Les autres sections ne sont pas concernées.

La fenêtre s'ouvre à un endroit étrange / la disposition semble décalée

La taille de la fenêtre et la position du séparateur sont mémorisées ; replacez-les et le prochain lancement les restaurera.
