FileInfo

User guide — select a file and instantly see everything known about it: streams and codecs for videos, tags and cover art for music, EXIF for photos, and Spotlight metadata for everything else.

Getting started

Installation

cd FileInfo
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
brew install ffmpeg exiftool

The interface

The window is split into two panels; the divider is draggable and its position is remembered:

  • File tree (left) — starts at your home folder. Use “Choose Folder…” to go anywhere; pick /Volumes for external drives.
  • Metadata panel (right) — loads automatically on selection, no button needed. Preview image on top (photo, video frame or album cover), then the file name, the filter box and the grouped data.
  • Filter box — narrows the list as you type: enter e.g. language and only language-related rows remain.
  • Right-click a row — “Copy value” / “Copy row”.
  • Language menu — switch the UI language (Hungarian, English, German, Spanish, French); takes effect after a restart.
  • SHA-256 button (bottom of the panel) — computes the selected file's checksum in the background, for any file size; the result is selectable and included in "Copy Metadata".
  • Dark and light mode — the app automatically follows the system appearance (System Settings → Appearance) and switches live; there is no separate toggle.
  • View → Show hidden files — like Finder, the tree hides dotfiles by default; toggle them here.

🎬 Video mp4mkvavimovwebm

FAQ / Troubleshooting

Why don't I see EXIF data for an image?

Screenshots and images downloaded or received via messaging apps usually carry no EXIF — services strip it. This is not a bug.

How do I reach a USB stick / external drive?

📁 Choose Folder… → pick the drive under Locations, or type /Volumes into the path bar.

Files on a network drive load slowly

Extraction speed depends on how fast the file can be read. The UI never freezes, and the data shown always belongs to the last selected file.

The Spotlight section is empty for some file

Spotlight doesn't index every location (temporary folders, some external drives). The other sections are unaffected.

The window opens in an odd place / layout looks off

Window size and divider position are remembered; drag them where you want and the next launch will restore them.
